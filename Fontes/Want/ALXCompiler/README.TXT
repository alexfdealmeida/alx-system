WANT is a Delphi-friendly build tool. See the file:

  doc/index.html 

for more information about WANT.

EXECUTING WANT

There's a copy of want.exe under the "bin" directory.

BUILDING AND TESTING WANT

Please use the make\bootstrap.bat script to build WANT for the first time. After
that, you can place a copy of bin\want.exe on a different directory, and use
that copy to continue building WANT.
